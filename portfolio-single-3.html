Project Description

The goal was to modernize its online presence and improve the user experience. I implemented a clean and contemporary design, using elegant colors that reflect the sophisticated atmosphere of the barbershop. Navigation was optimized to be more intuitive, and features such as online booking and a style gallery were added. Additionally, the site is now fully responsive, ensuring excellent viewing on any device.
